Introduction to Sensors

Units owned by a Decision Maker may be defined to have Sensors.  A Sensor detects "signals" emitted by the Attributes of other Units.  A Unit's attribute can be defined to emit a signal through a corresponding Emitter. 

There are two types of sensors available;  A "Global Sensor", that detects every possible attribute, and an "Attribute Sensor" that detects emissions from a single attribute.

Global Sensors have one parameter, a range.  Attribute Sensors have more complex Sensor Ranges.   


How to Define a Sensor

1. Right click on the "Sensors" entry on the Object Types tree to the left.

2. Select Create Sensor.  Enter a Sensor name in the resulting dialog.

3. Choose a "Global Sensor" or an "Attribute Sensor".

4. If an "Attribute Sensor" is chosen, you have the option of creating a Sensor Range for it.

Introduction to Classifications

A Classification is an enumerable string which can be applied to any asset within the DDD. Classifications defined here will be able to be assigned to assets via the DDD Client.

As a part of the scenario, you're also able to define a set of Classification Rules which enable the scenario to define different icons to be displayed based on the object's Classification.

How to Define a Classification

1. Right click on the "Classifications" entry on the Object Types tree to the left.

2. Select Create Classification.  Enter a classification name in the resulting dialog.

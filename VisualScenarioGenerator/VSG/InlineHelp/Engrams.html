Introduction to Engrams

Engrams are memory variables.  An Engram is composed of a name identifier and an initial value.  

The value of an engram can be changed during the course of a scenario run. Engrams can be of the following types:

  • String - A string is a literal list of characters.
  • Double - A double is a rational number, i.e. 3.14
  • Logical - A logical is either true or false.
 

An Engram's value can be "tested" when a Scenario Event is thrown.  

Uses of Engrams

Engram values can be very useful in influencing the behavior of Units or the direction of the Scenario as a whole.

How to Define an Engram

1. Right click on the "Engrams" entry on the Object Types tree to the left.

2. Select Create Engram.  Enter an Engram name in the resulting dialog.

3. Choose a type and an initial value for the Engram.

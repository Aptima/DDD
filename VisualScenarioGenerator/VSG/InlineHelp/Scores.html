Introduction to Scores

A Score is a set of Scoring Rules that are going to be applied to a set of Decision Makers.  When defining a Score it is necessary to first define one or more Scoring Rules.  

In the Score definition screen, select the Scoring Rules to apply, the Decision Makers to whom the score will apply, the Decision Makers who will see the score, and the initial value for the score.

How to Define a Score

1. Right click on the "Scores" entry on the Scoring tree to the left.

2. Select Create Score.  Enter an Score name in the resulting dialog.

3. Follow the onscreen instructions for defining a Score.

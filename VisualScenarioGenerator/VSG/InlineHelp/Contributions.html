Introduction to Contributions

A Contribution is used by a Combo Vulnerability to describe how much of a particular capability contributes to a desired state change.  

A contribution has the following parameters:

The capability to which the unit is vulnerable to.

The Intensity or amount of the capability that must be applied.

And two optional parameters; range and probability.  Range says the contribution must be applied within this distance.  Probability is the probability that if the assets are within range that the unit will be vulnerable.

   

How to Define a Contribution

1. Right click on the "Contributions" entry on the Object Types tree to the left.

2. Select Create Contribution.  Enter an Contribution name in the resulting dialog.

3. Follow the onscreen instructions for defining a contribution.

Introduction to Effects

Effects are used in conjunction with Proximities, proximities define the range of an effect.  An Effect is defined in terms of intensity, or magnitude, and an optional probability.  Probability dictates the "probability" of a capability successfully executing.   

How to Define an Effect

1. Right click on the "Effects" entry on the Object Types tree to the left.

2. Select Create Effect.  Enter an Effect name in the resulting dialog.

3. Choose an initial value for Intensity or Probability.

Introduction to Scoring Rules

Scoring Rules are the fundamental building blocks when defining Scores.  They enable the user to define the conditions that lead to a score being incremented or decremented.

Score Increment:  The amount by which the score is incremented if its conditions are met. (Can be negative).

Scoring Rule Type: There are two types of rules; Existence and State Transition.  

Existence rules are applied for every second that an object satisfies the rule's Who/What/Where conditions.  A State Transition Rule is applied when Object 1 causes Object 2 to transition states.

  • Who?:  The owner of the object is me, a friendly decision maker, or a hostile decision maker.
  • What?: The object is of any object, an object of a specific Unit Type or Unit.
  • Where?: The object is anywhere on the map, in a specifiec active region, or not in a specific active region.

Examples

Example 1:

Create a rule that decrements the score by 1 point for every second that an enemy unit is present in a no-fly-zone.

  • The user selects a Scoring Rule Type of Object_1_Exists and a Score Increment of -1.  
  • In the Object 1 group box:
    • Set the Who condition to Hostile, 
    • Set the What condition to Anywhere, 
    • Set the Where condition to In_Region (and then selects the region from a list of active regions).

 Example 2:

 Create a rule that awards a player 10 points every time a hostile aircraft is destroyed within a no-fly-zone.

  • The user selects a Scoring Rule Type of Object_2_State_Transition and a Score Increment of 10.
  • In the Object 1 group box:
    • Set the Who condition to Myselft.
    • Set the What condition to Any.
    • Set the Where condition to Anywhere.
  • In the Object 2 group box:
    • Set the Who condition to Hostile.
    • Set the What condition to Any.
    • Set the Where condition to In_Region (and then select the desired region).
    • Set the New State to Dead, and leave the From State Blank.

How to Define a Scoring Rule

1. Right click on the "Scoring Rules" entry on the Scoring tree to the left.

2. Select Create Scoring Rule.  Enter an Score name in the resulting dialog.

3. Follow the onscreen instructions for defining a Scoring Rule.

Introduction to Emitters

Emitters broadcast the current value of a Unit's Attribute.  A Unit may be defined to broadcast all of its attribute's values, or it may be defined to broadcast on a single attribute.

Choosing "Default", creates an Emitter that broadcasts the current value of all the Unit's attributes. Choosing "Invisible", creates an Emitter that broadcasts none of the Unit's attributes.  Choosing an attribute name creates an emitter that broadcasts the current value of the selected attribute.  Attribute Emitters have Levels.

How to Define an Emitter

1. Right click on the "Emitters" entry on the Object Types tree to the left.

2. Select Create Emitter.  Enter a Emitter name in the resulting dialog.

3. Choose a "Global Emitter" or an "Attribute Emitter".

4. If an "Attribute Emitter" is chosen, you have the option of creating a Level for it.
